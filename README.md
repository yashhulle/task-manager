# ✨ Task Manager Web App

A modern and responsive Task Manager built using **HTML, CSS, and JavaScript** with a clean UI and smooth user experience.

## 🚀 Features

* ➕ Add tasks easily
* ✅ Mark tasks as completed
* ❌ Delete tasks
* ✏️ Edit existing tasks
* 🔍 Filter tasks (All / Completed / Pending)
* 💾 Persistent storage using LocalStorage
* 🎨 Modern UI with neon glow and animated gradient borders
* 📱 Mobile app-style responsive design

## 🛠️ Tech Stack

* HTML5
* CSS3 (Animations, Glassmorphism, Gradient UI)
* JavaScript (DOM Manipulation, LocalStorage)

## ⚙️ How to Run

1. Clone the repository
2. Open `index.html` in your browser
